# Experimental Drug & Clinical Trial Explorer

This Streamlit application allows users to explore the latest experimental drug trials from around the world. Users can search by drug name, disease, company, or medical keyword, and view detailed information about each trial, including phase, status, sponsor, locations, and key dates. The app also displays recent related news articles for each selected trial.

## Features
- Search clinical trials by keyword
- View detailed information for each selected trial
- Automatically fetches related news articles
- Clean, simple, user-focused interface
